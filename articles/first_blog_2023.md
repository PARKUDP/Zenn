# CLIの手順
- Githubのリポジトリを作成します!
- Zennの記事や本を管理するフォルダ作成及びフォルダに移動！
- npm init --yes
    - 管理したいフォルダをデフォルト設定で初期化することができる。
- npm install zenn-cli
    - zenn-cliを入手することができる
- npx zenn init
    - README.mdやarticlesとbookという名前のディレクトリが作成されることを確認できる！
